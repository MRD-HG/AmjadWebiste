# Amjad Association Website

الموقع الرسمي لجمعية أمجاد للتربية والأعمال الاجتماعية (الصخيرات، المغرب). الهوية الموحدة:
- الاسم: جمعية أمجاد للتربية والأعمال الاجتماعية
- الشعار النصي: شعلة معارف.. جذور أمجاد
- السطر التعريفي: تطوع – تربية – إبداع – ريادة
- تميمة البرامج: «أصيل» يقود المحتوى الموجه للأطفال والتلاميذ

## الصفحات
- `index.html` – الصفحة الرئيسية: الهوية، البرامج الرئيسية، أصيل، آخر المستجدات، معرض الصور.
- `about.html` – من نحن: الرؤية، الرسالة، القيم، البرامج، معلومات التأسيس والترخيص.
- `programs.html` – خريطة البرامج: المدارس الذكية (إيكولوجية/اجتماعية/تحويل في 6 أشهر)، برنامج CAN 2025، برامج تربوية واجتماعية، ملف الرعاية.
- `services.html` – مجالات التدخل والشراكات: التأطير التربوي، المواكبة البيئية، المواكبة الاجتماعية، الرعاية والشراكات + نموذج طلب سريع.
- `news.html` – الأخبار: قصص مرتبطة بالبرامج مع شارات (المدرسة الذكية، CAN 2025، الرعاية).
- `contact.html` – الاتصال: معلومات رسمية، نموذج تواصل، خريطة الصخيرات.
- `js/main.js` – ترجمات أساسية، حركات بسيطة، رسائل نجاح للنماذج، ومعرض الصور.

## بنية الملفات
```
images/          # الشعار، الصور الداعمة، صور الأنشطة
js/main.js       # سكريبتات بسيطة للتفاعل والترجمة
resources/       # أصول إضافية (إن وجدت)
index.html ...   # بقية الصفحات الثابتة
```

## التشغيل محلياً
لا حاجة لبناء أو باك-إند:
1) افتح أي صفحة مباشرة في المتصفح (مثلاً: `index.html`).
2) إن أردت خادماً بسيطاً: `python -m http.server 8000` ثم زر `http://localhost:8000`.

## اعتماديات الواجهة
- Tailwind CDN للتنسيق السريع.
- Font Awesome للأيقونات.
- Leaflet (صفحة الاتصال) لعرض خريطة الصخيرات.

## ملاحظات المحتوى
- جميع الصفحات تستخدم الهوية الموحدة والنص التعريفي.
- البرامج المعروضة هي البرامج الرسمية: المدارس الذكية، CAN 2025 للأطفال والتلاميذ، الدعم التربوي/الاجتماعي، وملف الرعاية.
- alt للنصور مهيأ بالعربية لتحسين الوصول.

لأي استفسار أو تحديث، تفضلوا بالتواصل عبر البريد الرسمي: `contact@amjad-association.ma`.
